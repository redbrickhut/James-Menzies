Sound Library Batch Downloader

Written in: Java, JavaFX

Version 1.0

I wrote this utility for a friend in order to help streamline the process of downloading the BBC sound effect archive which contained in excess of 16000 files. The program will store information about the download in between sessions, and will organize them into a nested folder structure. It also utilizes the JAudioTagger library to tag the files so that they are convenient to search through in various music players. I hope to add more libraries as well as more user customization in a future version.

FX Auto-Complete Text Field

Written in: Java, JavaFX

Version 1.0

A simple control that I wrote to allow a user to select an object from a predefined list by inputting text. This control also uses autocomplete with camel-casing for speedy input for the end user. This control extends the TextField class, and has the additional property "selectedObject" for internal retrieval. A demo showcasing all of this is included in the repo.
